# Inspeção DevTools - Cards de Modalidades

## Objetivo

Encontrar as classes CSS exatas aplicadas nos cards de modalidades do site original através do DevTools.

## Método

1. Navegar para https://pontodobicho.com/jogo-do-bicho
2. Fazer scroll até os cards de modalidades
3. Inspecionar um card (ex: "Grupo") no DevTools
4. Identificar:
   - Classes CSS aplicadas
   - Valores de padding (px, py)
   - Valores de gap entre colunas
   - Font sizes
   - Border radius
   - Outras propriedades relevantes
